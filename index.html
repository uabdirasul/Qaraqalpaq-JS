Variable

Ózgeriwshiler

Hár qanday dástúr islewi ushın maǵlıwmat kerek.

Máselen hár birimiz kúndelikli paydalanatuǵın "Telegram" dástúri. Telegramdaǵı doslarımız atları, olar jazǵan xabarlar, "profıl" súwretleri bulardıń barlıǵı maǵlıwmat.

Ózgeriwshiler áyne usı maǵlıwmatlardı saqlawǵa járdem beredi.

Ózgeriwshi

Ózgeriwshi bul ózinde maǵlıwmat saqlaytuǵın “qutı”. Ózgeriwshilerge hár qıylı maǵlıwmatlardı saqlay alamız.

Tómendegi bir qatar kod message atlı ózgeriwshi jaratadı.

let message;

Endi bul ózgeriwshiniń ishine = operatorı arqalı maǵlıwmat saqlay alamız:

let message;
message = “Sálem”;

Al endi joqarıdaǵı tekst kompyuter yadına saqlandı hám ol ózgeriwshi menen baylanıstırıldı. Bul tekstten keyinshelik paydalanıw ushın ol menen biriktirilgen ózgeriwshiniń atınan paydalanamız:

let message;
message = “Sálem”;
alert(message) // ózgeriwshiniń ishindegi maǵlıwmattı kórsetedi.

Onı qısqartıw ushın ózgeriwshi jaratılǵan hám oǵan maǵlıwmat kiritilgen qatardı bir-birine qosıwımız múmkin:

let message = “Sálem”;

Jáne de birneshe ózgeriwshilerdi de bir qatarǵa jazıwımız múmkin:

let user = “Ibrayım”, age = 24, message = “Sálem”;

Bul qısqa kóriniwi múmkin, biraq tájiriybeli dástúrshiler onday qılıwdı máslahát etpeydi. Kod jazıwdaǵı tiykarǵı qaǵıydalardan biri - kodımız oqıwǵa ańsat bolıwı kerek bolǵanlıgı ushın hár bir ózgeriwshini bólek-bólek qatarlarǵa jazıw kerek.

Bólek qatarlarǵa jazıw úlkenirek kóriniwi múmkin, biraq ol oqıwǵa ańsat:

let user = “Ibrayım”;
let age = 25;
let mesasage = “Sálem”;

Ayırım dástúrshiler tómendegidey bólek qatarlarǵa jazıw usılınan paydalanadı:

let user = “Ibrayım”,
  age = 25,
  message = “Sálem”;

“comma-first” yaǵnıy birinshi útir usılınanda paydalanatuǵınlar bar:

let user = “Ibrayım”
  , age = 25
  , message = “Sálem”;

Texnik tárepten joqarıdaǵı usıllardıń barlıǵı bir jumıstı atqaradı. Olardan paydalanıw ózlerińizdiń talǵamıńız hám estetikańızǵa baylanıslı.

let’tiń ornına var

Eski skriptlerde, let gilt sózi ornına var degen gilt sózden de paydalanıp ózgeriwshi jaratıw múmkin edi:

var message = “Sálem”;

var gilt sózi menen let gilt sózi derlik birdey. var da ózgeriwshi jaratadı, biraq azǵantayózgesheligi bar.

Olar arasındaǵı parıqtı var ushın arnalǵan bólimde kórip shıǵamız.

Real ómirdegi uqsaslıq

Eger variable’dı ústine túkirarlanbas at jazılǵan qutı dep kóz aldimizǵa keltirsek, bul variable tusinigin arqayın túsinip alamız.

Mısal ushın: message atlı ózgeriwshini ishinde “Sálem!” degen jazıw bar, sırtına bolsa message dep jazılǵan qutı dep alatuǵın bolsaq:

Bul qutınıń ishine qálegen túrdegi maǵlıwmatımızdı qosa alamız.

Jáne de onı qálegenimizshe ózgerte alamız:

let message;
message = "Sálem!”;
message = “Dúnya”; // maǵlıwmat ózgerdi
alert(message);

Qutı ishindegi maǵlıwmat ózgertilse, eski maǵlıwmat óship ketedi:

Hám jáne, eki ózgeriwshi yáki “qutı” jaratıp, birinshisinen ekinshisine maǵliwmattı kóshirip de ótkere alamız:

let hello = “Sálem!”;
let message;
message = hello // hello ózgeriwshisinen maǵlıwmattı kóshirip message ózgeriwshisine ótkeredi // endi eki ózgeriwshiniń de maǵlıwmatı birdey
alert(hello);// Sálem
alert(message); // Sálem

Ekinshi márte qáte shıǵaradı

Ózgeriwshi let arqalı tek ǵana bir márte jazılıwı kerek.

Eki márte jazılsa ol qátelikke sebep boladı:

let message = “Sálem!”
// tákirarlanǵan let qátelik keltirip shıǵardı
let message = “Dúnya”; // SyntaxError: ‘message’ has already been declared

Ózgeriwshilerge at qoyıw

JavaScriptte ózgeriwshilerge at qoyıwda eki sheklew bar:

  1. Qoyılǵan at tek ǵana háripler, sanlar yáki $, _ ǵa uqsas belgilerden ibarat bolıwı kerek
  2. Qoyılǵan at sanlar menen baslanbawı kerek

Qoyıw múmkin bolǵan atlarǵa mısal:

let paydalanıwshıAtı;
let test123;

Eger qoyılǵan atımız bir neshe sózlerden ibarat bolsa camelCase tan paydalanıw máslahát etiledi. camelCase usılı bul - sózler izbe-iz jaylasadı, birinshi sózden tısqarı hámme sóz bas hárip penen baslanadı: judaUzınAt

$ hám _ belgilerinen de paydalanıwımız múmkin. Olar da háriplerge uqsaǵan, heshqanday máni bildirmeytuǵın, ápiwayı belgi. Tómendegi atlardan da paydalanıwǵa boladı:

let $ = 1;
let _ = 2;
alert($+_); // 3

Múmkin emes atlar:

let 1a;
let my-name; // Uncaught SyntaxError: Unexpected token '-’

Úlken menen kishiniń parqı bar

jas hám JAS - ekewi basqa-basqa ózgeriwshi etip qabıllanadı.

Tek ǵana Latınsha emes

Latın álipbesindegi háriplerinen basqa (Kirill, Qıtay logogramları) háripler menen de at qoyıwǵa boladı, biraq usınıs etilmeydi:

let имя = “…”;
let 我 = “…”;

Texnik tárepten bul jerde heshqanday qátelik joq, biraq xalıq aralıq qabıl etilgen konvenciyaǵa kóre ózgeriwshilerge tek ǵana inglis tilinde at beriw kerek.

Bul sózlerden paydalanıwǵa bolmaydı

Ózgeriwshige at qoyıwda paydalanıwǵa bolmaytuǵın sózlerdiń dizimi bar. Bul sózlerden paydalana almaymız, sebebi bul sózler JStiń ózin jaratıwda paydalanılǵan.

Máselen: let, class, return, function.

Tómendegi kod sintaksislik qátelik shıǵaradı:

let let = 5; // Uncaught SyntaxError: let is disallowed as a lexically bound name
let return = 5; // Uncaught SyntaxError: Unexpected token 'return’

use strict siz ózgeriwshi

Ádette ózgeriwshiden paydalanbastan aldın, onı gilt sóz benen jaratıp alıw kerek. Biraq eski versiyalarda onı gilt sózsiz, ápiwayı usılda da jaratıw múmkin edi. Eger eski kodlarǵa maslastırıw ushın use strict'siz jaratsaq, bul usıl ele de isleydi:

// bul use strict’siz usıl
num = 5;
alert(num); // 5

use strict ten paydalansaq, bul tájiriybesizlik penen jazılǵan kod - qátelik keltirip shıǵaradı:

“use strict”;
num = 5; // Uncaught ReferenceError: num is not defined

Konstantalar

Konstanta yaǵnıy maǵlıwmatın ózgertirip bolmaytuǵın ózgeriwshi jaratıw ushın let ornına const paydalanıladı:

const myBirthday = “17.04.2006”;

const arqalı jaratılǵan ózgeriwshiler konstantalar delinedi. Olardıń maǵlıwmatın ózgertiwdiń ilajı joq. Ózgertiwge háreket etilse qátelik keltirip shiǵaradı.

const myBirthday = “17.04.2006”;
myBirthday = “17.10.2006”; // Uncaught TypeError: Assignment to constant variable.

Eger dástúrshi jaratpaqshı bolǵan ózgeriwshi maǵlıwmatın keyinshelik ózgertpewin anıq bilse hám bunı basqa dástúrshilerge de bilmdirmekshi bolsa consttan paydalanıw kerek.

Bas háripli konstantalar

Uzın, yadlap qalıwǵa qıyın nárselerge laqap qoyıp, laqaptı bas hárip penen jazıp, const arqalı konstantalarǵa aylandırıp paydalanıw dástúrshiler ushın jaqsı ámeliyat esaplanadı.

Bunday konstantalarǵa bas hárip hám _ dan paydalanıp at qoyıladı.

Mısal ushın hexademical formatınan paydalanıp reń jaratıp kóreyik:

const COLOR_RED = "#F00";
const COLOR_GREEN = "#0F0";
const COLOR_BLUE = "#00F";
const COLOR_ORANGE = "#FF7F00";
// Usilardan bir ren’di saylaytug’in waqitta
let color = COLOR_ORANGE
alert(color) // #FF7F00

Bul usıldıń paydalı tárepleri:

  1. COLOR_ORANGE #FF7F00 ge qaraǵanda yadlap qalıwg’a ańsatlaw.
  2. di qáte jazıp alıwımız múmkin, biraq COLOR_ORANGE di qáte jazıw imkaniyatı kem.
  3. COLOR_ORANGE #FF7F00 ge qaraǵanda mazmunlıraq. COLOR_ORANGE di oqıǵanda onıń apelsin reń ekenin bile alamız.

Konstantlarǵa qashan tek bas hárip penen at qoyıw kerek hám qashan oǵan normal (qálegen belgi járdeminde) at qoyıw kerek?

Konstanta bul onıń maǵlıwmatı heshqashan ózgermesligin ańlatadı. Biraq, hexademical formatında jaratılǵan, qızıl reńge uqsaǵan - kod orınlanıwınan aldın maǵlıwmatı anıq bolǵan konstantalar yáki kod orınlanıwı waqtında maǵlıwmatı anıq bolatuǵın, biraq keyin ózgermeytuǵın konstantalar da bar.

Máselen:

// pageLoadTime = websayttıń júkleniw tezligi
const pageLoadTime = /* websayttı júklewge ketetuǵın waqıt */

pageLoadTime yaǵnıy websayttıń júkleniwine ketken waqıttı kod orınlanıwınan aldın bile almaymız. Sebebi, onı biliw ushın aldın websayt júkleniwi kerek, júkleniwi ushın kod orınlanıwı kerek. Demek oǵan qádimgidey, ózimiz qálegenimizshe at qoya alamız. Biraq, ol ele konstanta esaplanadı. Sebebi onnan keyin de onıń maǵlıwmatı ózgermeydi.

Basqasha qılıp aytqanda tek ǵana hard-coded yaǵnıy ózgeriwshi ishindegi maǵlıwmattı COLOR_ORANGEdegidey etip ózimiz qoldan jazsaq, oǵan bas háripler menen at qoyıw kerek.

Durıs at qoyiw

Ózgeriwshiler haqqındaǵı jáne bir áhmiyetli nárse bul, durıs at qoyıw.

Ózgeriwshi atı taza hám anıq bir mániske iye, ózi ishinde saqlaytuǵın maǵlıwmatqa túsindirme beretuǵın bolıw shárt.

Ózgeriwshilerge durıs at qoyıw eń áhmiyetli hám eń quramalı uqıp esaplanadı.

Ózgeriwshi atlarına bir kóz taslaw bul kodtıń úyrenshik dástúrshi yáki tájiriybeli dástúrshi tárepinen jazılǵanın ayta aladı.

Real proyektlerde kóp waqıt tazadan kod jazıwǵa emes aldınǵı dástúrshi jazǵan kodtı ózgertiw hám keńeytiwge ketedi. Durıs at qoyıw arqalı kóp waqıttan soń kod jazıwǵa qaytqanda da onı túsiniw ańsat boladı.

Ózgeriwshi jaratıwdan aldın oǵan eń mas keletuǵın at oylap tabıwıńız kerek. Usınday qılıw keyinirek sizge waqıttı tejewge járdem beredi.

At qoyıwda ámel qılınıw kerek bolǵan qaǵıydalar:

  1. Túsiniw ańsat bolǵan atlardan paydalanıń: userName (paydalanıwshı atı), shoppingCart (satıp alınǵan zat sebeti)
  2. Qısqartpa yáki a, b , c ǵa uqsaǵan júdá qısqa atlardan saqlanıń. Bolmasa, ne qılıp atırǵanıńızdı umıtıp qoyasız
  3. Atlardı maksimal xarakteristkalaytuǵın hám qısqa qılıń. Jaqsı emes atlarǵa mısal: data (maǵlıwmat), value (baha). Bunday atlar heshnárseni anıq túsindirmeydi. Bunday atlardı tek ǵana kod konteksti qaysı maǵlıwmat yáki bahaǵa silteme qılıp atırǵanın anıq aytsaǵana qollanıw kerek.
  4. Toparıńız ishindegi hám ózińiz jaratqan shártlerge ámel etiń. Eger saytqa kirgen adam user dep atalǵan bolsa, ózgeriwshi jaratıwda soǵan uqsas qılıp jaratıw kerek. Máselen: currentUser yáki newUser. currentUser hám user, bulardı user sózi biriktirip turıptı. Demek oǵan currentVisitor yáki newManInTown atları durıs kelmeydi.
