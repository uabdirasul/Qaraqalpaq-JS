Ózgeriwshiler

Hár qanday dástúr islewi ushın maǵlıwmat kerek.

Máselen hár birimiz kúndelikli paydalanatuǵın "Telegram" dástúri. Telegramdaǵı doslarımız atları, olar jazǵan xabarlar, "profıl" súwretleri bulardıń barlıǵı maǵlıwmat.

Ózgeriwshiler áyne usı maǵlıwmatlardı saqlawǵa járdem beredi.

Ózgeriwshi

Ózgeriwshi bul ózinde maǵlıwmat saqlaytuǵın “qutı”. Ózgeriwshilerge hár qıylı maǵlıwmatlardı saqlay alamız.

Tómendegi bir qatar kod message atlı ózgeriwshi jaratadı.

let message;

Endi bul ózgeriwshiniń ishine = operatorı arqalı maǵlıwmat saqlay alamız:

let message;
message = “Sálem”;

Al endi joqarıdaǵı tekst kompyuter yadına saqlandı hám ol ózgeriwshi menen baylanıstırıldı. Bul tekstten keyinshelik paydalanıw ushın ol menen biriktirilgen ózgeriwshiniń atınan paydalanamız:

let message;
message = “Sálem”;
alert(message) // ózgeriwshiniń ishindegi maǵlıwmattı kórsetedi.

Onı qısqartıw ushın ózgeriwshi jaratılǵan hám oǵan maǵlıwmat kiritilgen qatardı bir-birine qosıwımız múmkin:

let message = “Sálem”;

Jáne de birneshe ózgeriwshilerdi de bir qatarǵa jazıwımız múmkin:

let user = “Ibrayım”, age = 24, message = “Sálem”;

Bul qısqa kóriniwi múmkin, biraq tájiriybeli dástúrshiler onday qılıwdı máslahát etpeydi. Kod jazıwdaǵı tiykarǵı qaǵıydalardan biri - kodımız oqıwǵa ańsat bolıwı kerek bolǵanlıgı ushın hár bir ózgeriwshini bólek-bólek qatarlarǵa jazıw kerek.

Bólek qatarlarǵa jazıw úlkenirek kóriniwi múmkin, biraq ol oqıwǵa ańsat:

let user = “Ibrayım”;
let age = 25;
let mesasage = “Sálem”;

Ayırım dástúrshiler tómendegidey bólek qatarlarǵa jazıw usılınan paydalanadı:

let user = “Ibrayım”,
  age = 25,
  message = “Sálem”;

“comma-first” yaǵnıy birinshi útir usılınanda paydalanatuǵınlar bar:

let user = “Ibrayım”
  , age = 25
  , message = “Sálem”;

Texnik tárepten joqarıdaǵı usıllardıń barlıǵı bir jumıstı atqaradı. Olardan paydalanıw ózlerińizdiń talǵamıńız hám estetikańızǵa baylanıslı.

let’tiń ornına var

Eski skriptlerde, let gilt sózi ornına var degen gilt sózden de paydalanıp ózgeriwshi jaratıw múmkin edi:

var message = “Sálem”;

var gilt sózi menen let gilt sózi derlik birdey. var da ózgeriwshi jaratadı, biraq azǵantayózgesheligi bar.

Olar arasındaǵı parıqtı var ushın arnalǵan bólimde kórip shıǵamız.
